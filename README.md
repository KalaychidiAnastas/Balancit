Балансит — небольшой проект, который генерирует персональную "гарвардскую тарелку" (Harvard Plate) на основе текстового запроса пользователя. Проще говоря: вы отправляете предпочтения (например, калории, склонность к белку/жиру/углеводам), модель возвращает структуру тарелки, ингредиенты и рекомендации для пользователя. Изначально проект разрабатывался как прототип для стартапа с одноименным названием.
